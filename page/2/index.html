显示器校色技术

发表于 2024-08-10 更新于 2024-10-12 分类于 影视后期技术
本文字数： 1.6k 阅读时长 ≈ 6 分钟

满打满算也对百来台显示器进行过校色了，是时候整理下知识了。

为什么显示器需要校色？

“怎么手机和显示器颜色不一样？”

“怎么两台显示器颜色不一样？”

“甲方说颜色不对，你重新调一版”

为了避免以上问题，在严肃创作中，统一颜色标准是十分有必要的。

而各种屏幕的工艺、调教不同，所以显示器出厂颜色很难统一，具体体现在偏色、对比度过高、饱和度过高、亮度不准确等，需要后天使用专业的校色仪进行校色。

一.校色需要什么设备？

校色仪

品牌选择

常用的两个厂家：爱色丽、红蜘蛛

由于爱色丽校色仪可以直接将校色文件（.icc）写入操作系统中，属于硬件校色（存疑）,而红蜘蛛每次开机都需要打开软件并保持常驻后台才能校准屏幕，属于软件校色。因此我一般选择爱色丽校色仪进行校色。

爱色丽型号：

  • Colorchecker Display Pro
  • Colorchecker Display PlusHL：支持最新的MiniLED或者XDR，检测最高亮度可达10000cd检测（pro是3000）,能支持超高刷新。

二.校色流程

校色前注意事项

  1. 校色后会生成校色文件,文件的信息与显卡,显示器,系统,系统版本,绑定,所以校色完成后,如果更换显卡或者显示器则需要重新校色.

  2. 不同品牌不同批次显示器的材质,工艺不同,因此校色完有轻微差距是正常现象,如有强迫症可以购买同一批次的高端显示器.更贵的显示器一般色准也会更好,校色效果也会更好.检测是否校色正确应该用彩色图片对比,不可用纯白色,也是因为这个原因.

调色前要确定这3步：
01.显卡显示器恢复默认值，显卡在桌面空白处鼠标右键可以看到快捷入口(NVIDIA控制面板,所有选项都重置为默认,每个选项重置都要点击保存,3D需要重置两次)。

02.系统中显卡里没任何ICC（本是空白）

03.和颜色有关的软件，比如华硕奥创，护眼，虑蓝光等，有改变颜色或色温的软件要卸载。

校色仪无法识别

  • 卸载红蜘蛛驱动:
    • 如果用过红蜘蛛校色仪,需要先卸载红蜘蛛的驱动,爱色丽校色仪才可以识别
  • Xrite Device Service服务未开启:
    • 按住WIN键+R键，打开运行，在出来的命令窗口中输入“services.msc”打开服务策略组，往下拉找到Xrite Device Service服务，手动把它启动

Windows 10 台式电脑

1.打开显示器30分钟以上(以前的一些屏幕材质需要运行30分钟后才能稳定)
2.恢复显卡出厂值（笔记本和苹果是不需要），安装最新显卡驱动。
3.恢复显示器出厂值（显示器菜单上）

4.然后插上U盘，安装01软件，安装完之后，打开软件，等绿色勾，如不能打开，安装02软件,插入调色仪等待大约5-15分钟（看到提示3个绿色勾），然后选择【高级】就能按照册子对应的教程进行操作

lg27ul550(sRGB98%+IPS) 材质背投:白光LED，ICC版本2(显卡是Quadro等专业卡则需要选择ICC版本4)，颜色【中-211】

亮度老化后过不了120cd的亮度检测就要降低到100或者80

校色文件起名不能出现中文,会变乱码.

Mac笔记本电脑

  1. 打开设备30分钟
  2. 进入【偏好设置】–【显示器】（需去掉色彩还原，环境光的勾选）—【颜色】—选普通RGB描述。(M1只能选P3-500/600nits)
  3. 安装01文件夹的软件，如不能打开，则安装02文件夹软件
  4. 打开软件插入调色仪，等到绿色勾后，绿色勾上方选【高级】左边变化，左上角第一个【色彩管理】进去校色。
  5. 材质白光LED(看屏幕材质)，D65，sRGB(或者P3,看具体型号),本机—–ICC版本4（其他不用动）—–颜色（P3选【大】，2019年之前选【中】——进入校色，只勾亮度，调亮度。

校色完成后要记录标准亮度数据

四.重装系统后重新加载校色文件流程

  1. 桌面-右键-显示设置-显示-高级显示设置-显示器1的适配器属性-颜色管理-颜色管理-选择自己的显示器设备-使用我对此设备的设置-添加-浏览-选择自己的icc校色文件
  2. 桌面-右键-显示设置-显示-高级显示设置-显示器1的适配器属性-颜色管理-颜色管理-高级-设备配置文件(这里是指WIN自带看图的时候用哪个ICC，如果是双屏，因为只能用1个icc，另一个屏幕就会不对，下载Br就可以解决)-选择自己对应屏幕的icc校色文件-更改系统默认值-打勾使用Windows显示器校准

快门角度：

来自于以前的胶片机，适用于24、25、30帧的影片，50帧以上的影片可快门速度对动态模糊的影响不大了,可以用到360°，避免给曝光带来压力。

360度=所有部分都曝光=24帧下每张胶片曝光1/24S=快门速度1/24 ，更多运动模糊，梦幻效果
180度最常用：快门速度为帧率的两倍的倒数
90度：运动镜头拍摄，高速快门，更少运动模糊

什么是分镜？

分镜（Storyboard）：是在拍摄和制作影片之前，将我们脑中的成片，以「一次运镜」为单位进行分解，并标注上时长、景别和运镜方式等等信息的一个图表。（把你的文字版分镜给画出来，一个镜头一幅画）

制作分镜的好处？

可以更好地向摄影师表达自己的想法，降低沟通成本且如果是全特效的场景还需要结合其他方式向伙伴说明自己的拍摄想法；另外分镜也能让你的客户明白你想要的画面和创意、如果不满意就可以在分镜时期进行修改、修改成本较低。

如何去做一个分镜表？

首先要有创意梗概，然后细化创意、设计一个文字版的分镜（需要包含：
1.内景还是外景
2.时间，涉及「色温」
3.景别
4.镜头运动的方式
5.画面内容即演员的表演
6.场景
7.使用的道具
8.服装和化妆风格
9.这个镜头的时长
10.声音
