发表于 2023-09-01

HEXO常用命令

创建新文章 hexo new
sample： hexo new HEXO常用命令 ##如果标题包含空格，需要用引号括起来

发布文章三部曲： hexo clean ##清除缓存文件 (db.json) 和已生成的静态文件 (public)。在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。
hexo g ##generate 生成静态文件
hexo d ##deploy 部署到网站

本地预览服务器 hexo s
取消本地预览服务器 Control+C
